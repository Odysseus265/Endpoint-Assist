Overview

Endpoint Assist is a comprehensive IT help desk web application designed to streamline endpoint troubleshooting, network diagnostics, device management, and support ticket handling. Built with modern web technologies, it provides IT professionals with all the tools they need to efficiently support and maintain computer systems across Windows, macOS, and mobile platforms.

System Diagnostics

Monitor CPU, RAM, disk usage, and system health in real-time with visual gauges and detailed reports.

Network Tools

Ping tests, DNS resolution, traceroute, port checking, and WiFi diagnostics all in one place.

Security Status

Check Windows Defender status, firewall configuration, and get a comprehensive security score.

macOS Support

Troubleshooting guides for SMC reset, NVRAM, Safe Mode boot, and Disk Utility operations.

Mobile Devices

Manage iOS and Android devices with MDM enrollment, email config, and common issue resolution.

Ticket System

Create, track, and manage support tickets with categorization, priority levels, and status tracking.

Knowledge Base

Searchable library of IT solutions with step-by-step guides for common issues and organization-specific topics.

Asset Inventory

Complete hardware and software inventory with detailed specifications and license tracking.

Getting Started

System Requirements

  • Windows 10/11 operating system
  • Python 3.8 or higher
  • Modern web browser (Chrome, Firefox, Edge)
  • Administrator privileges (for some features)

Installation Steps

1

Install Dependencies

Run the following command to install required Python packages:

pip install flask flask-cors psutil wmi
2

Start the Application

Navigate to the application directory and run:

python app.py
3

Access the Dashboard

Open your browser and navigate to:

http://localhost:5001
Pro Tip

For the best experience, run the application with administrator privileges to access all system information.

Features Guide

Dashboard

The main dashboard provides a quick overview of your system's health, including real-time CPU, memory, and disk usage. It also displays recent tickets, security status, and network connectivity information.

System Health

Get detailed information about your operating system, processor, memory, and storage. Visual gauges show current usage levels, and the processes table displays top resource consumers.

  • Real-time CPU and memory monitoring
  • Disk space analysis with visual indicators
  • Battery status (for laptops)
  • Running processes overview
  • Boot time and uptime tracking

Security Status

Monitor your system's security posture with checks for Windows Defender, firewall status, and Windows Update status. The security score gives you a quick assessment of your protection level.

Network Diagnostics

Comprehensive network troubleshooting tools to diagnose connectivity issues:

  • Ping Test: Check if a host is reachable
  • DNS Resolution: Verify domain name resolution
  • Port Check: Test if specific ports are open
  • Traceroute: Visualize the path to a destination
  • WiFi Status: Check wireless connection details

Quick Tools

One-click solutions for common IT tasks:

  • Clean Temp Files: Remove temporary files to free disk space
  • Clear Browser Cache: Clear cached data from Chrome and Edge
  • Reset Network: Flush DNS and reset network configuration
  • View Error Logs: Check recent Windows error events

Ticket System

Create and manage support tickets with the built-in ticketing system:

  • Create tickets with title, description, category, and priority
  • Track ticket status (Open, In Progress, Resolved)
  • Filter tickets by status
  • Timestamp tracking for SLA compliance
Note

Tickets are stored in memory and will be cleared when the server restarts. For production use, consider integrating with a database.

Frequently Asked Questions

Why are some features showing "Unknown" or not loading?
Some system information requires administrator privileges to access. Try running the application as administrator by right-clicking on your terminal and selecting "Run as administrator" before starting the server.
Is my data secure?
All data is processed locally on your machine. No information is sent to external servers. The application runs entirely on localhost and only accesses local system information.
Can I use this tool remotely?
The current version is designed for local use. For remote access, you would need to configure the server to listen on your network interface and implement proper authentication and encryption.
How do I change the port number?
Open the app.py file and find the line app.run(host='0.0.0.0', port=5001, debug=True). Change the port number to your desired value.
Can I add custom tools or features?
Yes! The application is built with extensibility in mind. You can add new API endpoints in app.py and corresponding UI components in the templates and JavaScript files.
